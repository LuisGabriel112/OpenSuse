Interfaces

X window (comúnmente conocido como X11 o X) proporciona a los PCs Linux un entorno gráfico en el cual pueden ejecutarse distintos escritorios gráficos (Sin X window uno solamente podría tener un prompt ascii o entorno tipo shell en su PC Linux (es decir, únicamente trabajar el SO por línea de comandos)


En el X window de Linux, para obtener un entorno gráfico de usuario (GUI) amigable, encima de X window se ejecuta un Gestor de Ventanas o un Entorno de Escritorio. Es el caso de un usuario de openSUSE que típicamente está corriendo su PC en lo que se suele referir como nivel de ejecución 5 (En MacOS y en Windows no hay tal distinción entre shell ascii y el más nivel más alto del entorno gráfico, y por lo tanto MacOS/Windows tienen solamente un GUI, mientras que un usuario Linux puede elegir entre muchos).

Cuatro de los más populares escritorios están disponibles para openSUSE:


  • KDE
  • GNOME
  • Xfce
  • LXDE

Sin embargo, YaST 2 incluye la opción de descargar e instalar una amplia gama de entornos gráficos, entre los cuales se encuentran también:


  • Cinnamon
  • LXQt
  • Enlightenment
  • MATE
  • Deepin
  • Budgie

Cambiar de entorno gráfico en openSUSE solía no ser tan simple como en otras distribuciones de Linux, debemos pedirle a YaST que instale el nuevo entorno gráfico al mismo tiempo que desinstala el que tenemos actualmente, y puede ser un proceso un poco tardado. Una desventaja es que a veces pueden existir conflictos entre un entorno y otro, por lo que el nuevo entorno instalado podría tener bugs gráficos, para evitar esto, es una práctica común entre los usuarios de openSUSE el utilizar Snapper y sacar una imagen de su sistema antes de hacer un cambio de este tipo.

En sus ultimas versiones, el proyecto ha incorporado maneras mucho mas sencillas de llevar a cabo esta tarea, y basta con seleccionar la interfaz que querramos utilizar desde el menú de inicio de sesión
