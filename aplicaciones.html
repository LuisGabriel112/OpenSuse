Aplicaciones del sistema en la industria

El proyecto openSUSE cuenta con varias poderosas herramientas que se han vuelto sus principales atractivos a la hora de elegir un sistema operativo para trabajar, estas son:

Snapper

Snapper garantiza que no se produzcan problemas en el sistema incluso en caso de cambio. Esto se consigue a través de snapshots. Si se instala un software en openSUSE, se aplica una actualización o se cambia la configuración, se crea automáticamente un snapshot de este tipo. Se toman más snapshots a intervalos determinados. Así, si hay un problema o un cambio en el sistema va mal, el usuario puede simplemente volver a un estado anterior. Esto es especialmente ventajoso si quieres diseñar y optimizar la distribución de Linux según tus propias ideas. Con openSUSE, el usuario decide por sí mismo qué sistema es el mejor para sus propósitos. Para las empresas que necesitan contar con una infraestructura impecable sin sorpresas desagradables, openSUSE Leap es la elección correcta. Los desarrolladores que quieren estar siempre al día optan por openSUSE Tumbleweed, que al ser una versión que se renueva siempre viene con las últimas actualizaciones. También existen sistemas adecuados para otros fines Además del YaST ya mencionado, openSUSE proporciona otras herramientas útiles que facilitan mucho el trabajo con la distribución.

El sistema de imágenes KIWI puede crear dispositivos de servidor blindados o imágenes del disco duro y generar una máquina virtual.

El sistema automático de pruebas openQA comprueba todo el software y los componentes del sistema operativo y permite así un uso seguro.

También se incluye el Open Build Service (OBS), conocido también por otras distribuciones de Linux, que ayuda a la creación y distribución de paquetes de software. La herramienta fue desarrollada por la comunidad de openSUSE. Gnome, KDE, Xfce, LibreOffice, Mozilla Firefox y Opera también se incluyen en la instalación..

openSUSE Tumbleweed

openSUSE Tumbleweed es el buque insignia del proyecto y la base de las demás distribuciones. Las actualizaciones funcionales se proporcionan diariamente, de modo que la distribución está siempre al día. Un procedimiento consciente de pruebas garantiza el mantenimiento de la seguridad y la estabilidad. Por lo tanto, openSUSE Tumbleweed está dirigido principalmente a los desarrolladores.

openSUSE Leap

Con openSUSE Leap, los desarrolladores ofrecen desde 2015 una distribución gratuita que también vale para las empresas. El sistema funciona de forma estable y recibe actualizaciones de seguridad periódicas.

openSUSE MicroOS

Como su nombre indica, openSUSE MicroOS es un sistema operativo especialmente pequeño y minimalista. También se basa en openSUSE Tumbleweed, pero está pensado principalmente para el Edge Computing (informática que se lleva a cabo en o cerca de los usuarios o fuentes de datos, fuera del centro de datos / nube tradicional y centralizado.)

openSUSE Kubic

openSUSE Kubic es una evolución de openSUSE MicroOS y una plataforma de ordenador como servicio. Kubic incluye la herramienta de arranque del clúster Kubernetes kubeadm y también utiliza YaST.

openSUSE Medical

Un sistema operativo para usuarios del ámbito médico es openSUSE Medical. La distribución de Linux se basa en el antiguo sistema openSUSE 11.3 y está diseñada para gestionar los historiales de los pacientes, las recetas y otros datos en las consultas y clínicas.
